# 🎬 Movie Recommendation System

## 🚀 Live Demo
https://movie-recommendation-system1-7kt6.onrender.com/

---

## 📌 About Project
This is a **content-based movie recommendation system** built using Machine Learning.  
It recommends similar movies based on genres, cast, crew, and keywords.

---

## ⚙️ Tech Stack
- Python 🐍  
- Flask 🌐  
- Pandas 📊  
- Scikit-learn 🤖  
- NLTK 🧠  
- Docker 🐳  

---

## 🧠 How it Works
1. User enters a movie name  
2. System analyzes movie features  
3. Finds similar movies using similarity score  
4. Returns top 5 recommendations  

---

## 🎯 Example API
/recommend?movie=Avatar
